# Решение для Supplier Section - Генерация страниц товаров в КП

## Обзор решения

Создана удобная система для менеджеров по заполнению форм в "Supplier Section", которая позволяет автоматически генерировать страницы спецификации товаров в коммерческих предложениях (КП) в формате, аналогичном примеру "Example".

## Компоненты решения

### 1. Форма для менеджера (Supplier Section - Form)

**Расположение:** Левая часть дизайна

**Функциональность:**
- Позволяет менеджеру заполнить все необходимые данные о товаре
- Включает все поля, необходимые для создания спецификации в КП

**Поля формы:**

1. **Изделие*** (обязательное)
   - Описание основного продукта и его характеристик
   - Пример: "Лофт-перегородка раздвижная, стандарт. Профиль 40х20 мм, штапики 15х15 мм, порошковая окраска"

2. **Стекло*** (обязательное)
   - Характеристики стекла
   - Пример: "Закалённое тонированное бронза матовое, 6 мм"

3. **Размеры (мм)*** (обязательное)
   - Габариты изделия
   - Пример: "2500 х 4030"

4. **Цвет*** (обязательное)
   - Описание цвета и покрытия
   - Пример: "RAL 9005, муар — чёрный с микрофактурой"

5. **Комплектация**
   - Список включенных компонентов
   - Пример: "Ручки петли магниты направляющая ролики накладки уплотнитель заглушки ПСУЛ"

6. **Дополнительные услуги**
   - Доставка (₽): стоимость доставки
   - Монтаж (₽): стоимость монтажа
   - Замер (₽): стоимость замера

7. **Изображение товара (URL)**
   - Ссылка на изображение товара

**Кнопки действий:**
- **"Сгенерировать КП"** - создает финальную страницу спецификации
- **"Предпросмотр"** - показывает предварительный просмотр результата

### 2. Страница спецификации товара (Example - Product Specification Page)

**Расположение:** Правая часть дизайна

**Структура:**

**Левая колонка:**
- Бейдж "ФИКСИРОВАННАЯ ЦЕНА"
- Заголовок "Спецификация"
- Подзаголовок с описанием
- Детальная информация о товаре:
  - Изделие
  - Стекло
  - Размеры
  - Цвет
  - Комплектация
- Раздел дополнительных услуг:
  - Доставка
  - Монтаж
  - Замер

**Правая колонка:**
- Изображение товара (большое, 500px высота)
- Итоговая цена (крупным шрифтом)
- Примечание о НДС

**Дизайн:**
- Темный фон (#141414)
- Белый текст
- Четкая типографика
- Профессиональный внешний вид

## Как это работает

### Процесс генерации страницы:

1. **Заполнение формы**
   - Менеджер заполняет все поля в форме "Supplier Section"
   - Все обязательные поля (помечены *) должны быть заполнены

2. **Валидация данных**
   - Проверка заполнения обязательных полей
   - Проверка формата данных (размеры, цены)

3. **Расчет итоговой цены**
   - Суммирование всех услуг (доставка + монтаж + замер)
   - Форматирование цены с пробелами для читаемости

4. **Генерация страницы**
   - Создание HTML-страницы на основе шаблона "Example"
   - Подстановка данных из формы в соответствующие поля
   - Загрузка изображения товара по указанному URL

5. **Экспорт/Выгрузка**
   - Сохранение готовой страницы в формате HTML
   - Возможность экспорта в PDF для отправки клиенту

#### Вариант 1: Клиентская генерация (JavaScript)

```javascript
function generateSpecificationPage(formData) {
  // Получение данных из формы
  const productData = {
    product: formData.product.value,
    glass: formData.glass.value,
    dimensions: formData.dimensions.value,
    color: formData.color.value,
    комплектация: formData.комплектация.value,
    delivery: parseInt(formData.delivery.value) || 0,
    installation: parseInt(formData.installation.value) || 0,
    measurement: parseInt(formData.measurement.value) || 0,
    imageUrl: formData.imageUrl.value
  };
  
  // Расчет итоговой цены
  const total = productData.delivery + 
                productData.installation + 
                productData.measurement;
  
  // Форматирование цены
  const formattedTotal = total.toLocaleString('ru-RU');
  
  // Генерация HTML
  const html = generateHTML(productData, formattedTotal);
  
  // Создание и скачивание файла
  downloadHTML(html, 'specification.html');
}
```

## Преимущества решения

1. **Удобство для менеджера**
   - Простая и понятная форма
   - Все необходимые поля в одном месте
   - Предзаполненные примеры для понимания формата

2. **Автоматизация**
   - Автоматический расчет итоговой цены
   - Автоматическая генерация страницы
   - Минимизация ошибок при ручном вводе

3. **Консистентность**
   - Единый формат всех спецификаций
   - Профессиональный внешний вид
   - Соответствие брендбуку

4. **Эффективность**
   - Быстрое создание КП
   - Возможность сохранения шаблонов
   - История созданных спецификаций

## Дальнейшие улучшения

1. **Сохранение шаблонов**
   - Возможность сохранять часто используемые конфигурации
   - Быстрое создание новых спецификаций на основе шаблонов

2. **История и версионирование**
   - Сохранение истории изменений
   - Возможность отката к предыдущим версиям

3. **Интеграция с CRM**
   - Автоматическая привязка к клиентам
   - Отслеживание статуса КП

4. **Экспорт в разные форматы**
   - PDF для печати
   - Excel для расчетов
   - Email-шаблоны для отправки

5. **Мультиязычность**
   - Поддержка нескольких языков
   - Автоматический перевод спецификаций

## Использование

1. Откройте форму "Supplier Section"
2. Заполните все обязательные поля
3. Добавьте дополнительные услуги (если необходимо)
4. Укажите URL изображения товара
5. Нажмите "Предпросмотр" для проверки
6. Нажмите "Сгенерировать КП" для создания финальной страницы
7. Сохраните или отправьте готовую спецификацию клиенту
